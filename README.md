# 🍽️ Restaurant Tip Prediction using Linear Regression

This project predicts the **tip amount based on the total restaurant bill** using a **Linear Regression machine learning model**.

Built using **Python and Google Colab**.

---

## 📊 Dataset
The project uses the **Tips dataset** from the Seaborn library.

Features include:
- Total bill
- Tip amount
- Number of people at the table

---

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Scikit-learn
- Google Colab

---

## 🤖 Machine Learning Model
A **Linear Regression model** was trained to learn the relationship between:

Total Bill → Tip Amount

The model predicts the expected tip for any bill value.

---

## Example
Input:
Total Bill = 500

Output:
Predicted Tip ≈ 70–80
